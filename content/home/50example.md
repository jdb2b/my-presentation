> SV = (# * $ * %)/L

### Now Plug in some numbers...

|# Opps Created   |100     |    |             |    |
|-----------------|--------|----|-------------|----|
|$ Avg Deal Value |$5000   |    |Disqualified |20  |
|Avg Deal Length  |30 days |    |Won          |20  | 
|Sales Velocity   |$3333   |    |% Win Rate   |25% | 

<small>
% Win Rate = 20/(100-20) = 25%
<br>
Sales Velocity = ((100-20) * $5000 * 25%)/30 = $3333
</small>
